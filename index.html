M

Data-driven language for game developers

Get Started →

Simple

M is minimalist. Programs are built by combining few orthogonal building blocks. Its powerful type system frees programmers from declaring types, resulting in clear and readable code.

Flexible

M enforces data oriented design. All data is stored in components which allows designers to create entities with new behavior by combining different sets of components.

Tool

M complements your workflow. Use any editor that supports the language server protocol and generate gameplay code for Unity, Unreal or Godot.
