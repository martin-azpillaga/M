Project M

A data-driven programming language for game developers

Get Started →

Simple

A carefully crafted minimalist programming language with a built-in game programming standard library and implicit type inference.

Flexible

Use any editor with the language server protocol support like VS Code, and see the results in different engines. Unreal, Unity or Godot.

Performant

The code written in M is optimizable by default, runs in parallel across cores and can run in native code speed sometimes.
